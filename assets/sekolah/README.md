# Gambar sekolah untuk slider

Letakkan gambar rasmi, milik sendiri, atau gambar berlesen di folder ini.
Slider akan menggunakan fail `.jpg` berikut secara automatik:

```text

mrsm-kepala-batas.jpg
mrsm-johor-bahru.jpg
mrsm-tun-ghafar-baba.jpg

mrsm-kota-putra.jpg

mrsm-balik-pulau.jpg
sbpi-gombak.jpg
sekolah-tun-fatimah.jpg

sbpi-rawang.jpg
sekolah-alam-shah.jpg
kolej-melayu-kuala-kangsar.jpg
sekolah-seri-puteri.jpg
kolej-yayasan-saad.jpg
sms-hulu-selangor.jpg
```

Jika mana-mana fail belum ada, portal akan memaparkan ilustrasi fallback supaya
slider tidak rosak.
